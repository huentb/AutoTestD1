1. Cucumber using Serenity BDD and Screenplay

||Sử dụng ngôn ngữ Gherkin (Given/When/Then) viết specs
||Biến specs đó thành auto test
||Tự động tạo test report pass/fail

Ưu điểm: dễ hiểu, dễ tái sử dụng
Nhược điểm: setup phức tạp, nặng

Yêu cầu:
- Java JDK từ 1.8
- Maven 3 trở lên
- IDE Java: IntelliJ hoặc Eclipse
- Git

Cấu trúc project:
src
├───main
│   └───java
│       └───starter: chứa source code của hệ thống
└───test
    ├───java
    │   └───starter
    │       └───helpers (page-object, navigation, action): chứa class thực thi hành động open, input, submit,... 
    │       └───stepdefinitions: chứa code thực thi steps
    └───resources
        └───features: chứa specs
